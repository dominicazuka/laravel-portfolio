# Laravel Portfolio

This is my Laravel portfolio, which showcases my skills in Laravel development. The portfolio includes a variety of projects, such as a blog, a contact form, and a portfolio page. The code is well-documented and easy to understand.

## Technologies Used

* Laravel
* Tailwind CSS
* Alpine.js
* Axios
* Laravel Vite Plugin
* Lodash
